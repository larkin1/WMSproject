# SparkWMS
SparkWMS is a _very_ barebones WMS (Warehouse Management System)

## Features:
 - Add and remove items based on their location.
 - Can operate without wifi if necessary, and changes won't be lost
 - Commit based architecture.
 - Uses a supabase server on the backend.
 - not much more to it than that really.
